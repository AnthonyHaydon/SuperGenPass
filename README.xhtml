Android SuperGenPass utility

This is an Android implementation of SuperGenPass. In addition to using the form when you run the app, you can also use the "Share Page" menu item in the Browser to launch SGP and pre-fill the domain. Go to Menu → More → Share Page.

Download

  • download Android SuperGenPass v1.0.4

Source

The source for Android SGP is available via the Android SGP git repository. You can grab a copy for yourself by running:

git clone git://staticfree.info/git/SuperGenPass/
